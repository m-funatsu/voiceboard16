# VoiceBoard (ボイスボード) - AI搭載フィードバック収集・管理SaaS

## 概要

VoiceBoardは、SaaS運営者やプロダクトチーム向けに、ユーザーからのフィードバック（要望・バグ報告・改善提案）を収集・管理・分析するツールである。埋め込み可能なウィジェット、誰でも閲覧・投票できる公開ボード、AIによる類似フィードバックの自動統合と優先度スコアリングを備える。

## 推奨ランク: A

フィードバック管理市場は成熟しているが、Cannyの不透明な従量課金やUserVoiceの高額料金に対する不満が多い。フラット料金・投票者数無制限・無料枠でのAI機能提供で明確な差別化が可能。

## 料金プラン

- **Free**: $0/月 - 1ボード、50件/月、投票者無制限、基本AI
- **Pro**: $19/月 - 無制限ボード・投稿、フルAI分析、カスタムドメイン
- **Business**: $49/月 - ホワイトラベル、API、高度分析、優先サポート

## 技術スタック

- Next.js 16 + React 19 + TypeScript + Tailwind CSS 4
- Supabase (PostgreSQL + Auth + pgvector)
- Stripe (課金)
- OpenAI (text-embedding-3-small + GPT-4o-mini)

## 主要機能

1. **埋め込みウィジェット**: Shadow DOM JSスニペット + iframe埋め込み
2. **公開フィードバックボード**: 投稿・投票・ソート・フィルタ
3. **AI管理者ダッシュボード**: クラスタリング・優先度スコア・テーマサマリー
4. **管理画面**: プロジェクトCRUD・フィードバックモデレーション・アナリティクス・エクスポート
