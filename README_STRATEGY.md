# VoiceBoard（フィードバック管理SaaS） - 戦略ドキュメント

## 1. プロダクト概要 (逆引きPRD)
- **ターゲットユーザー**: SaaSプロダクトの運営者・PM（ユーザーフィードバックを体系的に収集・管理したいが、Canny等の既存ツールが高額な開発者・チーム）
- **解決する課題**: ユーザーの要望がSlack/メール/SNSに散在し、優先順位がつけられない。類似の要望を手動で集約する手間が大きい
- **ソリューション**: 埋め込みウィジェットでフィードバックを収集し、公開ボードで投票を受け付け、AIが類似フィードバックを自動クラスタリング・優先度スコアリング。Supabase＋Stripe連携の本格SaaS構成
- **コア機能**:
  - 埋め込みフィードバックウィジェット（JS/iframeでサイトに設置、テーマ・位置・カラーのカスタマイズ）
  - 公開フィードバックボード（slug付きURL、誰でも閲覧・投票可能、アカウント不要のフィンガープリント認証）
  - AI自動クラスタリング（embedding+コサイン類似度による類似フィードバック統合、閾値0.85）
  - 優先度スコア自動算出（投票数x2.0 + 新しさスコアx10.0 + クラスタサイズx1.5）
  - ステータス管理（open/planned/in_progress/completed/declined）
  - Supabaseバックエンド（認証・RLS・リアルタイム同期）
  - Stripe課金統合（Free/Pro/Businessの3プラン）

## 2. Stripeマネタイズ戦略
- **現状**: Stripe実装済み（`@stripe/stripe-js`・`stripe`パッケージ導入済、`/api/stripe/checkout`・`/api/stripe/status`エンドポイント、planId指定によるpro/business切り替え）
- **無料プラン制限**: 1ボード・月50件フィードバック・基本AI（類似検出）・VoiceBoardブランド表示
- **プレミアム機能**:
  - Pro（¥1,900/月 = $19）: 無制限ボード・無制限フィードバック・フルAI分析・カスタムドメイン・ブランド非表示
  - Business（¥4,900/月 = $49）: Proの全機能＋ホワイトラベル・APIアクセス・高度な分析・優先サポート
- **推奨価格**: ¥1,900/月（Pro）を主力プランとして推進。投票者数は全プラン無制限を差別化ポイントに
- **ペイウォール配置**: 2ボード目の作成時、月51件目のフィードバック受信時、フルAI分析機能へのアクセス時、ブランド非表示設定時

## 3. リテンション施策
- **メール通知トリガー**:
  - 新しいフィードバックが投稿された際のリアルタイム通知
  - 投票数が急増したフィードバックの「トレンド」アラート
  - 週次ダイジェスト（新規フィードバック数・人気トップ3・ステータス変更サマリー）
  - フィードバックのステータスが「completed」に変更された際のフォロワー通知
- **データ蓄積フック**: フィードバック・投票データが蓄積されるほどAIクラスタリングの精度が向上し、優先度スコアの信頼性が増す。公開ボードのURLを共有済みのユーザーは移行コストが高い（URLの変更が困難）
- **習慣形成**: フィードバックボードのURLをプロダクト内に常設。ユーザーが投票するたびに通知が届き、PM/開発者が定期的にダッシュボードを確認するサイクルを構築

## 4. 不足機能 (Quick Wins)
1. **管理者向けAI分析ダッシュボード** - クラスタリング結果の可視化・優先度ランキング・カテゴリ別トレンドグラフ。LPでは「AI管理者ダッシュボード」を謳っているが、ダッシュボードは基本的な統計のみで、AI分析結果の表示画面が未実装
2. **フィードバックへの公式回答機能** - 管理者がフィードバックに対して「検討中」「対応予定」等の公式コメントを投稿できる機能。ユーザーとの対話を促進し、エンゲージメントを向上
3. **Slack/Discord連携** - 新規フィードバック投稿時や投票数閾値到達時にSlack/Discordに自動通知。開発チームのワークフローにフィードバック確認を自然に組み込む
